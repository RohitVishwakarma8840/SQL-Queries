### **Types of SQL Commands**

1. **DDL (Data Definition Language):**
   - Used to define and modify the structure of database objects such as tables, schemas, and indexes.
   - Examples of DDL commands: `CREATE`, `ALTER`, `RENAME`, `TRUNCATE`, `DROP`.

2. **DQL (Data Query Language):**
   - Used to retrieve data from a database.
   - Example of DQL command: `SELECT`.

3. **DML (Data Manipulation Language):**
   - Used to manipulate the data stored in the database.
   - Examples of DML commands: `INSERT`, `UPDATE`, `DELETE`.

4. **DCL (Data Control Language):**
   - Used to manage access rights and permissions in the database.
   - Examples of DCL commands: `GRANT`, `REVOKE`.

5. **TCL (Transaction Control Language):**
   - Used to manage transactions in a database to ensure consistency.
   - Examples of TCL commands: `START TRANSACTION`, `COMMIT`, `ROLLBACK`.
